# @oliver139/eslint-config

Oliver's ESLint config preset powered by `@antfu/eslint-config`

## Project Setup

```sh
pnpm install
```

### Run src/index.ts

```sh
pnpm start
```

### Build the project with Rollup

```sh
pnpm build
```

### Build the project with Tsc

```sh
pnpm build:tsc
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
